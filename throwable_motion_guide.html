3. 2-Tier Application + Skeleton Mapping

  • Grenade follows 2-Tier: Tier 1 = character locomotion pool (BlendSpace BS_*) + Tier 2 = throw action (ThrowHigh/Low). Uses ABP (anim state machine).
  • Smoke / Flash / deploy-types are Montage-based (*_M_1P_Montage).
  • 1P hands / high-detail   3P world / others' view
  • Equipment-use actions (Fire/GetWeapon/Put) → each device's own skeleton
  • Character throw / locomotion → NV_DF BaseMale skeleton

4. Throw / Deploy Mechanism

  • Throw type (grenade / smoke): play Throw/Fire anim → at AnimNotify, detach hand mesh + spawn projectile → flight / explosion
  • Deploy type (claymore / mine): FireStart (begin) → FireLoop (deploying) → FireEnd (done = place _Static mesh in world) → CancelFire
AnimNotify exact timing & projectile class need to be confirmed from the DF gameplay BP (e.g. BP_AbilityBullet) or coordinated during implementation.
